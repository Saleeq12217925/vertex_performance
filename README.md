# Fitness Centre

A modern, responsive gym landing page built with React and Vite. This project showcases services, pricing, a gallery, and membership information with fast static deployment readiness.

## Features

- React 19 + Vite 8 application
- Responsive layout for desktop and mobile
- Fast static site output via `vite build`
- Deployment-ready for Vercel, Netlify, and other static hosts

## Local development

```bash
npm install
npm run dev
```

Open `http://localhost:5173` in your browser.

## Build for production

```bash
npm run build
```

The compiled site is generated in the `dist/` folder.

## Preview production build

```bash
npm run preview
```

## Vercel deployment

1. Go to https://vercel.com and sign in.
2. Click **New Project** → **Import Git Repository**.
3. Select your GitHub repo: `https://github.com/Saleeq12217925/vertex_performance`
4. Use the following settings:
   - Build command: `npm run build`
   - Output directory: `dist`
5. Click **Deploy**.

No additional Vercel configuration is required for this project.
